    Bicing network
    This graph represents the network of Bicing stations based on mutual nearest neighbourhood, it is interactive and the user can look for any station and get information about it and its neighbourhood. This can be done through the search bar or by clicking on the nodes. There is also available a grouping tool based on graph modularity partition.
    From this graph we can extrapolate the existance of eighteen unique classes or clusters. It can also be seen that there are certain nodes with high betweeness centrality, this information can be translated as the most centrical positions, the better connected ones.
    Stations centrality
    On this map we can see the betweeness centrality of all stations represented by its size and color. The bigger and darker a node gets, the more centrality it has. The betweness centrality is the ratio between all possible shortest paths of two nodes such that another one is on this path and all shortest paths between both nodes (most centered nodes).
    This map shows us that Provença, Còrsega and Passeig de Gracia are the central stations of the network, all located in the financial district of Barcelona.
    This information has been obtained from the previous graph.
    Clusters
    Based on the modularity partition of the network previously displayed, we have performed convex hull for each class.
    This map shows us that each modularity partition class suites the layout of a district of Barcelona. We can confirm that all stations are evenly distributed to fulfill the layout of the city.
